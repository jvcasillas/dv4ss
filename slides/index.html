Grammar of graphics

What you need to know

Wilkinson (2012)

We are interesting in mapping
data to aesthetics

What is data?

  • a column(s) in a dataframe
    (excel, .csv, .tsv, etc.)

Aesthetics?

  • a visual property of a plot,
    i.e., position, shape, color, etc.

ggplot2 overview

  • Written by Hadley Wickham when he was a graduate student at Iowa State

  • Automatically deals with spacings, text, titles but also allows you to annotate by ‚Äúadding‚Äù

  • Plots are built up in layers

  • Plot the data, overlay a summary, meta data and/or annotation

Components

  • Works with a dataframe

  • aesthetic mappings: how data are mapped to coordinates, color, size, shape, etc.

  • geoms: geometric objects like points, lines, boxplots, etc.

  • stats: statistical transformations

  • facets: ‚Äòpanels‚Äô for conditional plots

A template‚Ä¶

ggplot(
  data = DATA_FRAME, 
  mapping = aes(AESTHETIC)
) + 
  geom_xxx()
  • DATA_FRAME will be replaced with your dataframe
  • AESTHETIC will be replaced with the aesthetic mappings of your choice
  • geom_xxx will be replaced with a specific geom, i.e., geom_point()

Histogram

  • Beginner
  • Intermediate
# Continuous variable on the x-axis
ggplot(
  data = new_data, 
  mapping = aes(x = score)
) + 
  geom_histogram() 

# Streamline code and adjust options
ggplot(new_data) + 
  aes(x = score) + 
  geom_histogram(
    color = "black", 
    fill = "grey70", 
    bins = 9
  ) + 
  theme_bw(
    base_size = 30, 
    base_family = "Palatino"
  )

Scatterplot

  • Beginner
  • Intermediate
# Continuous variables on both axis
ggplot(
  data = new_data, 
  mapping = aes(x = proficiency, y = score)

ggplot(new_data) +

A better way‚Ä¶ point estimate + spread

  • Beginner
  • Intermediate
# Point estimate can be mean, median, mode, 
# Spread can be SD, SE, 95% CI, etc.
ggplot(
  data = new_data, 
  mapping = aes(x = group, y = score) 
) + 
  stat_summary(
    fun.data = mean_sdl, 
    geom = 'pointrange', 
    size = 2, 
    linewidth = 2
  ) 

# Customize, add caption
ggplot(new_data) + 
  aes(x = group, y = score) + 
  geom_point(alpha = 0.3, size = 4) + 
  stat_summary(
    fun.data = mean_cl_boot, 
    geom = 'pointrange', 
    size = 4, 
    linewidth = 2
  ) + 
  labs(caption = "Mean ¬±95% CI") + 
  theme_minimal(base_size = 30)
